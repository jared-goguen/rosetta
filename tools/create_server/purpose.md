Scaffold a new MCP server project with convention-based tool structure, test harness, and stdio transport.

This tool creates a complete, runnable MCP server project following Rosetta conventions. It generates all the boilerplate files needed for a new MCP server including package.json, TypeScript configuration, entry points, test infrastructure, and optional initial tool directories.

The generated project will have:
- Standard MCP server structure with stdio transport
- Convention-based tool scanning from tools/ directory  
- Complete test infrastructure with subprocess harness
- TypeScript configuration optimized for MCP development
- All necessary dependencies for MCP server development
